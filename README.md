# chat_bot
Chat-bot for homework
## Python Chat Bot (Flask)

### Учебный проект по разработке чат-бота на Python с веб-интерфейсом.
Бот обрабатывает базовые пользовательские запросы, выполняет математические операции и сохраняет историю диалогов.

Проект разработан в рамках изучения Python, Flask и работы с пользовательским вводом.

### Возможности

#### 1. Общение с ботом

Бот умеет распознавать базовые пользовательские намерения:
	•	приветствия
	•	прощания
	•	ответы на стандартные вопросы (FAQ)

Примеры:
'''
привет
здравствуйте
что ты умеешь?
как тебя зовут?
'''

#### 2. Математические операции

Бот умеет выполнять простые математические вычисления.

Поддерживаются:
##### Бинарные операции
'''
2+2
10-5
6*7
12/3
2^3
8**2
'''

##### Квадратный корень
'''
sqrt 16
√16
'''

##### Работа с отрицательными числами
'''
-5+10
-5-5
'''

##### Обработка ошибок
Бот корректно обрабатывает некорректный ввод:
'''
12/0 → На ноль делить нельзя
sqrt -16 → нельзя извлечь корень из отрицательного числа
'''

#### 3. Сохранение истории чата
История сообщений сохраняется в JSON файл:
storage/chat_[chat_id].json
Это позволяет сохранять диалог между пользователем и ботом.

#### 4. Пользовательские настройки

Бот поддерживает простые пользовательские настройки.

##### Изменение имени
'''сменить имя на [Имя]'''
После этого бот будет обращаться к пользователю по новому имени.

##### Изменение языка
'''сменить язык на en'''
Функциональность находится в разработке.

### Запуск проекта
1. Установка зависимостей
'''pip install -r requirements.txt'''
2. Запуск сервера
'''python main.py'''

После запуска приложение будет доступно по адресу: http://127.0.0.1:5000

### Интерфейс

Приложение имеет простой веб-интерфейс:
